新建

新建项目会在电脑端创建目录，写入 CodeX 桌面项目列表，并立即开启一个 CodeX 会话。

使用帮助

1. 下载和配置 CodeX CLI

  1. 在 Windows 上安装 Node.js LTS，并确认 PowerShell 能运行 node -v 和 npm -v。
  2. 安装或更新 CodeX CLI，保证命令行里能运行 codex.cmd。
  3. 登录 CodeX 账号，并先在电脑端确认 CodeX 可以正常打开项目目录。
  4. 启动本机 app-server：codex.cmd app-server --listen ws://127.0.0.1:4500。

2. 通过贝锐蒲公英配网

  1. 在 Windows 电脑和 iPhone 上登录同一个蒲公英账号或同一个组网。
  2. 确认手机能看到电脑设备，并记下电脑的蒲公英虚拟 IP。
  3. Windows 防火墙允许 Node.js 或网关端口被蒲公英网络访问；默认端口是 8787，可在 .env 的 PORT 中修改。
  4. 手机和电脑无需暴露公网，只通过蒲公英虚拟局域网访问。

3. 启动 AIToPhone 网关

  1. 在本项目目录运行 powershell -ExecutionPolicy Bypass -File .\scripts\restart-server.ps1。
  2. 脚本会输出可访问链接，其中蒲公英/Tailscale/局域网地址都可以用于手机访问。
  3. 如果网页显示未连接，先确认 CodeX app-server 和 AIToPhone 网关都在运行。

4. 获取网页链接

  1. 优先使用脚本输出的完整链接，格式类似 http://电脑虚拟IP:端口/?token=访问口令。
  2. 在 iPhone Safari 打开该链接，确认能进入聊天页后可添加到主屏幕。
  3. 链接生成后通常不需要再改；只有端口、虚拟 IP 或访问口令变化时才需要重新获取。

5. 项目和对话

  1. 项目和对话会由电脑端网关秒级自动同步，并和 CodeX 桌面端项目列表保持一致。
  2. 选择项目后点击「新建」，即可在对应目录开启 CodeX 对话。
  3. 历史对话会保存在本机 data/ 目录，不会提交到 GitHub。
