# AIToPhone Web 测试清单

本文档按专业前端验收视角整理，覆盖手机端 Web、电脑端网关、假数据同步和异常场景。自动化假数据测试可运行：

```powershell
npm run test:web
```

## 1. 页面加载与鉴权

- 无 token 访问：应打开连接设置，不泄露项目列表。
- URL 带 token 访问：应进入聊天页并保存本机登录态。
- token 错误：接口返回 401，页面提示重新输入访问口令。
- Service Worker 缓存：`index.html`、`app.js`、`styles.css` 版本号一致，更新后手机能拿到新资源。
- 断网/网关关闭：页面应显示事件连接断开，并自动重连。

## 2. 移动端布局

- iPhone Safari 竖屏：顶部、消息区、输入栏不重叠。
- iPhone Safari 横屏：消息区和输入栏仍在可视区域。
- 添加到主屏幕后打开：安全区、底部输入栏、侧边栏高度正常。
- 输入法弹出：页面不自动放大，输入栏上移，消息区可见。
- 输入法收起：页面恢复默认大小和位置，`keyboard-inset` 归零。
- 输入框字号不低于 16px，避免 iOS 聚焦自动缩放。

## 3. 侧边栏项目与对话

- 项目列表过长：项目区域可独立滑动。
- 点击项目标题/小三角：项目设为选中态，并展开/收起对话列表。
- 折叠状态刷新后保留。
- 选中项目样式清晰，且同一时间只有一个项目选中。
- 新建对话：手机端立即出现新对话，电脑端 5 秒内同步。
- 删除对话：手机端立即消失，电脑端 5 秒内同步。
- 删除项目：手机端隐藏，CodeX 桌面项目状态同步清理。
- 项目名称不显示本地绝对路径。

## 4. 聊天输入与展示

- 输入文本后点击发送：用户气泡立即显示。
- AI 未开始回答前：显示 `思考中...`。
- AI 第一段回答到达：移除 `思考中...` 并显示真实回答。
- 发送失败：移除临时用户气泡和思考占位，显示错误提示。
- 连续发送：发送按钮不会卡死，第二次可继续点击。
- Markdown：代码块、列表、链接、行内代码正常展示。
- 长文本：气泡不溢出屏幕，消息区自动滚到底。
- 中文、英文、数字、标点、换行都能正常显示。

## 5. 附件与文件

- 点击加号：可选择图片/文件。
- 图片上传：显示附件预览，发送后路径进入消息体。
- 文件上传：显示文件 chip，发送后可下载。
- 多附件：顺序、名称、大小处理正常。
- 上传失败：显示错误，不清空未发送文本。
- 文件下载接口：只能访问 uploads 或允许项目根目录内文件。

## 6. 同步与实时性

- WebSocket 首次连接：收到 `sync` 帧后更新项目、对话、账户状态。
- WebSocket 断开：2.5 秒左右自动重连。
- 兜底轮询：4 秒左右拉取一次 `/api/config`。
- 服务端同步：默认 3 秒一轮，不被账户额度接口阻塞。
- `message` 帧：立即更新当前对话。
- `turn-complete` 帧：恢复发送按钮，移除思考占位。
- 重复帧：同一 message id 不重复显示。
- 旧 seq 帧：不覆盖新状态。

## 7. 账户与额度

- 账户读取成功：显示账户、计划、同步时间。
- 额度读取成功：显示剩余额度、5h 剩余、7天剩余、累计 tokens。
- 账户接口慢：项目和对话同步不被阻塞。
- 账户接口失败：保留已有状态或显示错误，不影响聊天。

## 8. 兼容性与性能

- iPhone Safari。
- iPhone 主屏幕 PWA。
- Windows Edge/Chrome。
- 桌面宽屏：左侧栏固定，聊天区宽度正常。
- 手机窄屏：侧边栏覆盖式打开，遮罩可关闭。
- 100 条消息：滚动顺畅。
- 多项目、多对话：侧边栏不卡顿。

## 9. 安全与隐私

- Git 提交不包含 token、内网 IP、公网 IP、上传文件、data 会话。
- 访问口令不显示在页面主 UI。
- 下载文件路径必须被限制在允许目录内。
- 删除项目不删除磁盘文件，只清理 AIToPhone/CodeX 项目列表状态。

## 10. 本轮自动测试覆盖

`npm run test:web` 使用假数据验证：

- 前端资源版本和 Service Worker 缓存一致。
- iOS 键盘/viewport 防缩放关键配置存在。
- 发送后立即显示用户气泡和 `思考中...` 的代码路径存在。
- 项目折叠选中态逻辑存在。
- 客户端 4 秒兜底轮询、2.5 秒重连、服务端 3 秒同步存在。
- 假手机端创建线程、发送用户消息和附件。
- 假电脑端重新加载后能看到手机端创建的对话与消息。
- 删除线程后消息同步删除。
- CodeX 增量消息、完成事件、非最终事件的标准化解析。
